# Welcome to MinID Elements

Minid Elements is a modern web component library, primarily for MinID but also designed to be used by anyone. It provides accessible, customizable UI components styled with Tailwind CSS, and works with any framework or plain HTML.

## Compatibility with Designsystemet

MinID Elements is fully compatible with the Digitaliseringsdirektoratets [Designsystemet](https://github.com/digdir/designsystemet). This ensures that your applications have a consistent look and feel, regardless of the components you use.

## Available Components

This library offers a wide range of components to cover your UI needs. Explore the Storybook to see them in action and learn how to use them.
