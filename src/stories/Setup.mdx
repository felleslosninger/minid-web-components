# Styling with Tailwind

MinID elements provides a Tailwind theme based on[Designsystemet.](https://www.designsystemet.no/)
You can use to style your components.

## Installation

See the [Tailwind documentation](https://tailwindcss.com/docs/installation) for information on how to install. We recommend PostCSS if you are using Custom Components/Lit Components.

```html
<p class="text-body-md">Hello on you</p>
```
